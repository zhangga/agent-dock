# Profile v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a minimal Profile v1 export/import loop that wraps the current Backup stack, previews changes, and applies valid imported skills into Backup.

**Architecture:** Add a pure `profileStore` core module for schema creation, parsing, preview, and apply decisions. Server routes read/write the active stack and delegate profile logic to the core module. The existing single-page UI adds profile controls to the Backup tab and refreshes existing Backup/Restore state after import.

**Tech Stack:** TypeScript, Node.js standard library, existing HTTP server, existing HTML string UI, Vitest.

---

## File Structure

- Create `src/core/profileStore.ts`: Profile v1 schema, parsing, preview, and merge helpers.
- Create `tests/core/profileStore.test.ts`: Core TDD coverage.
- Modify `src/server/server.ts`: Add profile export, import preview, and import apply endpoints.
- Modify `tests/server/server.test.ts`: API tests and UI HTML tests.
- Modify `src/ui/page.ts`: Add Backup tab profile controls and browser-side handlers.
- Modify `README.md`: Document profile export/import workflow.
